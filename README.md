## 📝 Описание

**HTML Cleaner Pro** - это приложение с графическим интерфейсом для очистки HTML кода веб-страниц от:
- CSS классов и стилей
- JavaScript кода
- Мета-тегов и линков
- Нежелательных тегов (`div`, `span`, `iframe` и др.)

Программа сохраняет семантическую структуру HTML, удаляя только визуальные и поведенческие элементы, оставляя чистый контент.

## ✨ Особенности

### 🎯 Основные возможности

- ✅ **Очистка от CSS** - полное удаление классов, стилей и CSS-атрибутов
- ✅ **Удаление JavaScript** - очистка от скриптов и событий
- ✅ **Гибкая настройка** - выбор элементов для удаления
- ✅ **Предпросмотр результата** - просмотр очищенного кода до сохранения
- ✅ **Сохранение в UTF-8** - корректная работа с русским языком и спецсимволами
- ✅ **Подсветка синтаксиса** - удобное отображение HTML в предпросмотре

### 🛠️ Дополнительные функции

- 🗑️ Удаление тегов `<div>` и `<span>`
- 🔗 Сохранение ссылок (`href`)
- 🖼️ Сохранение изображений (`src`)
- 📋 Вставка URL из буфера обмена
- 💾 Сохранение результатов в файл
- 📁 Открытие папки с результатами
- ⌨️ Горячие клавиши
- 🌐 Автоопределение кодировки (UTF-8, Windows-1251, и др.)

### 🎨 Интерфейс

- Современный дизайн
- Интуитивно понятное управление
- Статус бар с информацией о процессе
- Индикатор прогресса
- Поддержка темной темы в предпросмотре

## 💻 Требования

- **Python** 3.7 или выше
- **Операционная система**: Windows / Linux / macOS
- **Библиотеки**:
  - `requests` - для загрузки страниц
  - `beautifulsoup4` - для парсинга HTML
  - `tkinter` - встроен в Python (для GUI версии)
